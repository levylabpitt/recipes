# Graphene Contacts Recipes

This directory contains recipes for making electrical contacts to graphene devices.

## Available Recipes

- **ti-au-standard.md**: Standard Ti/Au contacts via photolithography and liftoff
  - Geometry: Hall bar (6 contacts)
  - Metal stack: Ti (5nm) / Au (30nm)
  - Method: Photolithography + e-beam evaporation
